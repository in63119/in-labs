## 왜 App Router?

폴더 기반 라우팅과 서버 컴포넌트로 초기 로딩과 개발 생산성을 동시에 잡을 수 있다.

### 필수 파일

- `layout.tsx`: 공통 레이아웃
- `page.tsx`: 페이지
- `route.ts`: API 라우트
- `loading.tsx`: 로딩 UI
